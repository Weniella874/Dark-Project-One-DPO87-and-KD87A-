# Dark-Project-One-DPO87-and-KD87A-
Программное обеспечение DPOne для клавиатур Dark Project (DPO87 и KD87A), переработанная версия!


Данное ПО было создано одним человеком,
по ссылке "https://t.me/+2rz_URQ2DbsxOTY6" можно вступить в группу разработчика данной модификации!


1. Все референсы кнопок и текстур находятся в папке "uires\image"
2. Настройки перевода находятся в файле "lang_cn.xml" по пути uires\Translator\
3. В папке xml можно изменить оснывные настройки интерфейса, описание файлов приведенно ниже:

---------------------------------------------------------
XML Файлы:

1. color_panel.xml - Цветовая панель в меню LED
2. dlg_main.xml - Главное окно
3. dlg_tray.xml - Окно в трее
4. kbpanel.xml - Окно настройки эффектов и ползунков
5. kbside.xml - Само меню LED
6. keyboard.xml - Клавиатура в меню LED
7. macro.xml - Меню настройки макросов
8. menu_more.xml - Доп меню доступное по нажатию на стрелку сверху в правой части окна
9. menu_tray.xml - Меню в трее при нажатии на окно правой кнопкой мыши (кнопка выход)
10. sys_xml_msgbox.xml - Меню системных кнопок
---------------------------------------------------------


ПРЕДУПРЕЖДЕНИЕ: ПО ПРОВЕРЕННО ТОЛЬКО НА КЛАВАТУРАХ DARK PROJECT СЕРИЙ DPO87 и серии DARK PROJECT ONE, ДРУГИЕ УСТРОЙСТВА РАБОТАЮЩИЕ НА ПО DPOne НЕ БЫЛИ ПРОВЕРЕННЫ С ДАННОЙ КАСТОМИЗАЦИЕЙ!
Если вы хотите поучавствовать в развитии проекта обратитесь в группу разработчика данной кастомизации!


---------------------------------------------------------

Дополнительная информация о клавиатуре DARK PROJECT DPO87

- VID: 0x1A2C
- PID: 0x1513

---------------------------------------------------------

- Контроллер (MCU)
- Производитель: Sonix
- Семейство: SN32 (очень вероятно)
- Предположительные модели: SN32F248, SN32F268
- Используется кастомная прошивка Dark Project
- Закрытый протокол подсветки

---------------------------------------------------------

- Подсветка (ВАЖНО)
- Не QMK
- Не VIA
- Не стандартный HID RGB
- Не OpenRGB «из коробки»

Как реально управляется: Через vendor-specific HID, Через большие пакеты (527 байт), С инициализацией режима,
Используется тот же протокол, что и в SignalRGB Без точного бинарного формата клавиатура игнорирует команды.
