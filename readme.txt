1.usb vid и pid-код
 Откройте конфигурационный файл и найдите <usb..../> Элемент, измените в нем vid и pid, если вы хотите игнорировать vid или pid, то введите 0, если вы хотите игнорировать интерфейс и вспомогательное устройство, то введите -1

2.Изменить тип языка
 Откройте конфигурационный файл и найдите <local..../>Элемент, измените значение языка в нем на "cn" или "en".

3.Изменять/добавлять/ удалять список и ключевые значения мультимедийных ключей
 Откройте конфигурационный файл и найдите <mediakey>...</mediakey> элемент, измените/добавьте/удалите дочерний элемент <ключ.../>, где name - это отображаемое содержимое списка, а code - ключевое значение

4.Изменять/добавлять/удалять список и ключевые значения функциональных клавиш
 Откройте конфигурационный файл и найдите <функциональный ключ>...</functkey> элемент, измените/добавьте/удалите дочерний элемент <ключ.../>, где name - это отображаемое содержимое списка, а code - ключевое значение,
 Код может представлять собой комбинацию нескольких ключей и значений, но должен быть разделен английскими запятыми","

5.Добавить перевод
 Перевод на английский: По умолчанию используется EN, поэтому нет необходимости добавлять
 Перевод на китайский: Откройте uires\Translator\lang_cn.xml-файл в <контексте>...Добавьте соответствующие элементы перевода между </context>, источник - это оригинальный текст, перевод - это перевод

6.Кнопки изменения/добавления/удаления
 Открытые позиции.XML-файл модифицируется в соответствии с соответствующим форматом. Rect - это диапазон ключей, id - значение ключа, color - цвет фона ключа, а имя может быть проигнорировано.
